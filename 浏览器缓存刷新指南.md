# 解决方案：强制刷新浏览器缓存

## 问题原因
浏览器缓存了旧版本的HTML和JS文件，导致翻译未生效。

## 解决方法

### 方法1：强制刷新（推荐）

在浏览器中打开 http://localhost:4700/，然后按：

**Mac用户**:
- `Cmd + Shift + R` (Chrome/Safari)
- `Cmd + Option + R` (Firefox)

**Windows用户**:
- `Ctrl + Shift + R` (Chrome/Edge)
- `Ctrl + F5` (Firefox)
- `Ctrl + Fn + Shift + R` (某些笔记本电脑)

### 方法2：清除缓存后刷新

1. 打开开发者工具 (F12 或右键→检查)
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 方法3：无痕模式测试

1. 打开新的无痕窗口
   - Chrome: `Cmd + Shift + N` (Mac) / `Ctrl + Shift + N` (Windows)
   - Safari: `Cmd + Shift + N` (Mac)
2. 访问 http://localhost:4700/
3. 查看中文翻译是否生效

### 方法4：手动清除浏览器缓存

**Chrome**:
1. 地址栏输入: `chrome://settings/clearBrowserData`
2. 选择"缓存的图片和文件"
3. 点击"清除数据"

**Safari**:
1. 菜单栏 → Safari → 偏好设置 → 隐私
2. 点击"管理网站数据"
3. 删除 localhost 的数据

## 如果还是没变化

1. **确认URL正确**: 访问 http://localhost:4700/ (不要用https://)
2. **检查端口**: 确认是4700端口，不是其他端口
3. **重启服务器**:
   ```bash
   # 停止服务器
   # 重新启动
   ZUZU_DATA_DIR=$HOME/.虫族 npm run dev
   ```
4. **清除所有浏览器数据**: 包括cookies、缓存、存储
